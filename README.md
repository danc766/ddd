## Описание приложения

Данное мобильное приложение разрабатывается на Flutter и предназначено для удобного планирования задач и управления временем. С его помощью пользователь сможет создавать задачи, добавлять описание и устанавливать сроки выполнения.

Приложение решает проблему забытых дедлайнов и неорганизованности, с которой часто сталкиваются студенты и люди с большим количеством дел. Оно помогает структурировать учебные, рабочие и личные задачи в одном месте.

Целевая аудитория приложения — студенты, школьники и молодые специалисты, которые хотят повысить свою продуктивность и эффективно планировать своё время с помощью мобильного устройства.
